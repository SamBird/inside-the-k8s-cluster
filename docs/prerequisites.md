# Prerequisites

## Required local tooling

- Docker runtime available on your machine
- `kubectl` configured to access local clusters
- `kind` binary on PATH
- `make` and `bash`
- Python 3.11+ for backend
- Node.js 20+ and `npm` for frontend

## Recommended versions

- Kubernetes client (`kubectl`) v1.29+
- kind v0.22+
- Python 3.11+
- Node.js 20+

## Verification commands

```bash
docker version
kubectl version --client
kind version
make --version
bash --version
python3 --version
node --version
npm --version
```
